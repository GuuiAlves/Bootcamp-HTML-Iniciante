Tags básicas em HTML

  1. Tag 'head'

    2. Quando inserimos a tag head em nosso HTML, essa tag é responsável para criar o cabeçalho daquela pagina web.

  2. Tag 'title'

    3. Aprendi que a tag title é utilizada para definir o nome daquele site ou uma referencia principal de nomemclatura dele.

  3. Tag 'body'

    4. Essa tage define o corpo do site, todo conteudo entre o cabeçalho até o rodape daquela página.

  4. Tag 'h'

    5. Utilizamos essa tag para definir o Título ou Sub título de algum tipo de conteudo em texto que iremos apresentar em sequencia.

  5. Tag 'p'

    6. Tag p utilizada para inserirmos o conteudo de texto após de definir o título.

Tags de listagem

  1. Tag 'ol'

    2. Utilizada para definir uma lista ordenada de itens com uma sequencia numeral,

    1. Item 1
    2. Item 2
    3. Item 3

  2. Tag 'ul'

    3. Utilizada para criar uma lista não ordenada de itens, sem uma sequencia numeralexemplo abaixo:

    • Item 1
    • Item 2
    • Item 3
